📖 **Project Description**

Customer churn is a critical challenge for businesses, especially in banking and telecom sectors. This project aims to predict whether a customer will leave the service (churn) or stay, using a Deep Learning model (Artificial Neural Network - ANN).

The model is trained on customer data such as credit score, geography, gender, age, balance, and other financial attributes. Data preprocessing techniques like encoding, feature scaling, and train-test splitting are applied to prepare the dataset.

A multi-layer neural network is built using TensorFlow/Keras to capture complex patterns in customer behavior and make accurate predictions.

🎯 **Objective**

Predict customer churn (Exited = 1 or 0)
Help businesses identify at-risk customers
Improve customer retention strategies

🚀 **Features**

Data Cleaning & Preprocessing
Encoding (Label + One-Hot Encoding)
Feature Scaling using StandardScaler
ANN Model with multiple hidden layers
Model evaluation using accuracy & confusion matrix
Visualization using Seaborn & Matplotlib
